# TEAMIFY PROJECT (BACKEND)

**TEAMIFY project** is the API I created as a soccer fan and lover. This entertaining project aim to make football lovers real actors of their favorite sport.

**TEAMIFY** as the following goals :
Allow fans to create and build their own soccer team, to have a view upon their team value to control prices and see how the competitve team that they are dreaming of would cost to the club they would manage.

_Please note that all contribution from you will be greatly appreciated._

**IMPLEMENTED FEATURES**

- Find a specific team in the whole europeean 6 big soccer leagues
- Find players all players existing, in activities or retired
